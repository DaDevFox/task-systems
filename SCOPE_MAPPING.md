# SCOPE to Directory Mapping

This file maps the short-hand SCOPE identifiers used in commit messages to their corresponding project directories in the monorepo.

- `INV_`: `inventory-core`
- `WKFL`: `home-manager` (soon to be `workflow-core`)
- `SHRD`: `shared`
- `TASK`: `tasker-core`
- `USER`: `user-core`
- `ALL_`: All projects
